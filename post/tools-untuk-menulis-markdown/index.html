Menggunakan ssg (static site generator) memang mewajibkan kita mengerti markdown , untuk menulis menggunakan markdown tak sesulit yang aku bayangkan dulu , jikab saya pakai blogger dulu nulis bisa langsung di sediakan di dasboard , atau bila kita ingin menulis langsung html juga sudah ada fitur nya di blogger

Aku saat Membuat artikel ini menggunakan android , karna gak punya laptop , dengan terminal menggunakan termux , untuk menulis markdown cukup mudah dan gak perlu memerlukan banyak alat

Di artikel ini aku akan sharing aplikasi untuk menulis markdown di android , ada banyak tools yang bisa kamu gunakan yang pertama

  1. Stackedit.io (Online)

Kamu bisa Menulis markdown secara online di stackedit.io , fitur yang disediakan pun lengkap , kamu bisa cek sendiri di website nya . saya tidak memakai layanan ini karena online dan kuota internet saya terbatas heheh

  1. https://jbt.github.io (Online)

Sama Seperti Stackedit , dengan website ini kamu juga bisa membuat/mengimpor tulisan kamu ke dalam web jbt ini untuk mengedite tulisan mu.

Lalu saya Nulis Menggunakan apa? Saya menggunakan Text editor decoder , karna aplikasi tersebut sudah mendukung markdown . untuk kalian yang penasaran juga dengan aplikasi ini silahkan kunjungi di playstore atau lewat link berikut ini https://play.google.com/store/apps/details?id=com.paprbit.dcoder

Itu Merupakan Alat untuk kita menulis , lalu untuk belajar markdown dimana ya ? kalau saya belajar markdown di dokumentasi https://daringfireball.net/projects/markdown/ , sudah banyak kok yang membagikan ilmu tentang markdown ini bisa kalian searching aja di google
